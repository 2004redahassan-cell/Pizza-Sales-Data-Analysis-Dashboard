# Pizza Sales Data Analysis Dashboard 🍕📊

## 📌 Project Overview
This project presents an interactive **Power BI Dashboard** built to analyze sales performance for a pizza restaurant. The goal is to provide actionable insights into revenue, order trends, product popularity, and customer behavior to optimize inventory and sales strategies.

## 🛠️ Tech Stack Used
* **Data Analysis:** SQL (for data extraction, cleaning, and KPI verification)
* **Data Visualization:** Power BI (DAX formulas, Slicers, Interactive Charts)
* **UI/UX Design:** Custom professional layout and color palette

## 💡 Key Insights & Features
* **Executive KPIs:** Real-time tracking of Total Revenue ($817.9K), Total Orders (21K), Total Pizzas Sold (50K), Average Order Value ($38.31), and Average Pizzas per Order.
* **Sales Trends:** Identification of peak order days (Fridays & Saturdays) and monthly variations.
* **Product Performance:** Breakdowns by Pizza Category (Classic, Supreme, Chicken, Veggie) and Pizza Size to highlight maximum contributors to revenue.
* **Top Sellers:** Dynamically filters Top 5 Best Sellers by Revenue (e.g., The Thai Chicken Pizza, The Barbecue Chicken Pizza).
* **Advanced Navigation:** Interactive sidebar navigation with dedicated tabs for **Home** views and **Product Insights**.

## 📂 Project Structure
* `Pizza_Sales_Dashboard.pbix` - The core Power BI dashboard file.
* `SQL_Queries.sql` - SQL script containing data cleaning and aggregation analysis.
* `Data/` - Folder containing the dataset (CSV/Excel format).
